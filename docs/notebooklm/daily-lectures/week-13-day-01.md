## 2. Mini lecture
The week focus is Google Course 1: Foundations of Cybersecurity, and today's objective is kick off the week's topic and build the right vocabulary. The main terms in scope are CIA Triad and Risk. The focus is on what each one does, how it operates, and how it affects the rest of the system.

CIA Triad is security model built around confidentiality, integrity, and availability. It provides a basic framework for judging what a control is protecting or what a failure affected. Mechanically, it evaluates whether data stays secret, unaltered, and reachable, then uses those goals to choose and review controls. Risk is possibility that a threat will exploit a weakness and cause harm to an asset or process. It helps teams prioritize which security problems deserve treatment first. Mechanically, it combines the chance of a harmful event with the likely impact so work can be ranked and resourced. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
