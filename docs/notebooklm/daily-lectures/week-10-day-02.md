## 2. Mini lecture

The main terms in scope are Access Point and SOHO Router. The focus is on what each one does, how it operates, and how it affects the rest of the system.

Access Point is network device that bridges wireless clients onto a wired LAN. It provides Wi-Fi coverage so wireless devices can join the local network. Mechanically, it advertises a network name, authenticates clients, and forwards their traffic between radio channels and the wired switch path. SOHO Router is integrated gateway device common in small office and home networks. It combines routing, switching, wireless access, NAT, and firewall features in a small-network gateway. Mechanically, it combines routing, NAT, DHCP, switching, and wireless control into one device for small local networks.
