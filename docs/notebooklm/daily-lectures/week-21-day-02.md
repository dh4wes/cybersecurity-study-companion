The main terms in scope are Escalation Path and Case Management. The focus is on what each one does, how it operates, and how it affects the rest of the system.

Escalation Path is defined process for raising incidents to appropriate teams and leadership. It routes incidents to higher authority or specialist teams using predefined severity, impact, and responsibility criteria. Mechanically, it routes an issue through defined responders and thresholds so decisions move quickly when severity or scope increases. Case Management is tracking and coordination of investigation tasks, evidence, and decisions. It maintains case status, assigned work, evidence references, and decisions throughout an investigation lifecycle. Mechanically, it keeps alerts, evidence, notes, and task status in one record so investigations remain coordinated across analysts. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
