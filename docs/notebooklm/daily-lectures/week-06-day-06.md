# Weekly synthesis: LAN, WAN, OSI Model

## 2. Mini lecture
This is the review day for Networking foundations: models, media, devices, and topologies. The goal is to connect the week's main ideas into one working model rather than introduce a new topic. The core terms are LAN, WAN, OSI Model, TCP/IP Model, Network Media, and Network Interface Card.

Across the week, the key mechanisms were: LAN: uses local switching and addressing so devices exchange frames directly across a contained network segment WAN: uses routed links across carrier or internet paths so separate local networks can exchange traffic over distance OSI Model: breaks communication into layers so encapsulation, addressing, transport, and applications can be analyzed one function at a time TCP/IP Model: organizes communication into link, internet, transport, and application roles so packet flow can be traced across the stack The value of the review is in seeing how these pieces connect across setup, operation, failure, and troubleshooting.
