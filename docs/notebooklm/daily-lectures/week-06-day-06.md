## 4. Key distinctions and common confusions
- LAN vs WAN — LAN: Local area network that connects devices across a limited physical space such as a room, office, or campus. WAN: Wide area network that connects distant sites across provider infrastructure or large geographic areas.
- WAN vs LAN — WAN: Wide area network that connects distant sites across provider infrastructure or large geographic areas. LAN: Local area network that connects devices across a limited physical space such as a room, office, or campus.
- OSI Model vs TCP/IP Model — OSI Model: Seven-layer reference model used to describe how network communication functions. TCP/IP Model: Practical layered model used to describe how internet protocols work together.
- TCP/IP Model vs OSI Model — TCP/IP Model: Practical layered model used to describe how internet protocols work together. OSI Model: Seven-layer reference model used to describe how network communication functions.
- Network Media vs Copper Cabling — Network Media: Physical transmission medium that carries network signals, such as copper, fiber, or radio. Copper Cabling: Network cabling medium that carries data as electrical signals over metal conductors.
