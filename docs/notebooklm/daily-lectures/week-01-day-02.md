# Motherboard, Chipset, and PSU (Power Supply Unit) in context

## 1. Today’s concept in plain language

## 2. Mini lecture
The week focus is A+ Core 1: hardware, boot process, and internal components, and today's objective is deepen concepts and connect them to the bigger system. The main terms in scope are Motherboard, Chipset, and PSU (Power Supply Unit). The focus is on what each one does, how it operates, and how it affects the rest of the system.

Motherboard is main circuit board that connects CPU, memory, storage, and expansion components. It provides power distribution, bus pathways, firmware interfaces, and sockets connecting the main system components. Mechanically, it connects CPU, memory, firmware, storage, and buses so components can exchange power, timing, and data signals. Chipset is controller logic on the motherboard that coordinates communication between the processor, storage, USB, and expansion interfaces. It routes data and control signals between major subsystems that do not connect directly to the CPU. Mechanically, it links processor lanes, storage controllers, USB paths, and peripheral buses so board components can exchange data predictably. PSU (Power Supply Unit) is power supply unit that converts incoming AC power into regulated DC outputs for computer components. It delivers stable power rails to the motherboard, storage, fans, and expansion devices. Mechanically, it converts wall power to DC, regulates voltage on several rails, and distributes the correct connectors to each component. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
