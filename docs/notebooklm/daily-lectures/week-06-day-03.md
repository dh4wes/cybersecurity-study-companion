Network Media means physical transmission medium that carries network signals, such as copper, fiber, or radio. Its practical job is that it provides the path that network frames or bits travel across between interfaces. Mechanically, it encodes data as electrical, light, or radio signals and carries that signal across a cable or wireless channel. Network Interface Card means hardware adapter that gives a device a physical or wireless connection to a network. Its practical job is that it provides the interface that sends and receives frames or signals on the chosen medium. Mechanically, it converts device data into network signaling on the local medium and receives incoming traffic for the host. Copper Cabling means network cabling medium that carries data as electrical signals over metal conductors. Its practical job is that it provides common local connectivity for endpoints, switches, and other nearby network devices. Mechanically, it sends encoded electrical pulses across twisted conductors so connected interfaces can transmit and receive bits.
