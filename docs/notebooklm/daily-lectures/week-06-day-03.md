## 2. Mini lecture
The week focus is Networking foundations: models, media, devices, and topologies, and today's objective is apply the concepts with examples, drills, or comparisons. The main terms in scope are Network Media, Network Interface Card, and Copper Cabling. The focus is on what each one does, how it operates, and how it affects the rest of the system.

Network Media is physical transmission medium that carries network signals, such as copper, fiber, or radio. It provides the path that network frames or bits travel across between interfaces. Mechanically, it encodes data as electrical, light, or radio signals and carries that signal across a cable or wireless channel. Network Interface Card is hardware adapter that gives a device a physical or wireless connection to a network. It provides the interface that sends and receives frames or signals on the chosen medium. Mechanically, it converts device data into network signaling on the local medium and receives incoming traffic for the host. Copper Cabling is network cabling medium that carries data as electrical signals over metal conductors. It provides common local connectivity for endpoints, switches, and other nearby network devices. Mechanically, it sends encoded electrical pulses across twisted conductors so connected interfaces can transmit and receive bits. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
