Network Media means physical transmission medium that carries network signals, such as copper, fiber, or radio. Its practical job is that it provides the path that network frames or bits travel across between interfaces. Mechanically, it encodes data as electrical, light, or radio signals and carries that signal across a cable or wireless channel. Network Interface Card means hardware adapter that gives a device a physical or wireless connection to a network. Its practical job is that it provides the interface that sends and receives frames or signals on the chosen medium. Mechanically, it converts device data into network signaling on the local medium and receives incoming traffic for the host. Copper Cabling means network cabling medium that carries data as electrical signals over metal conductors. Its practical job is that it provides common local connectivity for endpoints, switches, and other nearby network devices. Mechanically, it sends encoded electrical pulses across twisted conductors so connected interfaces can transmit and receive bits. ## 4. Key distinctions and common confusions
- Network Media vs Copper Cabling — Network Media: Physical transmission medium that carries network signals, such as copper, fiber, or radio. Copper Cabling: Network cabling medium that carries data as electrical signals over metal conductors.
- Network Interface Card vs MAC Address — Network Interface Card: Hardware adapter that gives a device a physical or wireless connection to a network. MAC Address: Hardware identifier used by network interfaces on local segments.
- Copper Cabling vs Fiber Optic — Copper Cabling: Network cabling medium that carries data as electrical signals over metal conductors. Fiber Optic: Network medium that carries data as pulses of light through glass or plastic strands.
- Recognition vs diagnosis — knowing the term name is not enough; know what evidence would make you pick it in a scenario.
