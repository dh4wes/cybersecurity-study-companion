## 2. Mini lecture
This is the review day for Google Course 6: Detection and response (part 1). The goal is to connect the week's main ideas into one working model rather than introduce a new topic. The core terms are Alert Triage, False Positive, Indicator of Compromise, Incident Severity, Containment, and Eradication.

Across the week, the key mechanisms were: Alert Triage: checks source fidelity, correlates nearby telemetry, and assigns severity so analysts queue real threats ahead of benign noise False Positive: appears when a rule matches benign activity because indicators, thresholds, or context are too broad Indicator of Compromise: uses observable artifacts such as hashes, domains, or registry changes to match activity against known malicious patterns Incident Severity: combines impact, scope, and urgency into a rating that drives escalation, communication, and response effort The value of the review is in seeing how these pieces connect across setup, operation, failure, and troubleshooting.
