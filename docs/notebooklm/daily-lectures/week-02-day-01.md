## 2. Mini lecture
The week focus is A+ Core 1: mobile devices, networking basics, wireless, and virtualization, and today's objective is kick off the week's topic and build the right vocabulary. The main terms in scope are Mobile Device, Laptop, and Docking Station. The focus is on what each one does, how it operates, and how it affects the rest of the system.

Mobile Device is portable endpoint such as a phone, tablet, or lightweight computer with integrated radios and battery power. It delivers computing, connectivity, and local apps in a form factor built for movement and short sessions. Mechanically, it combines processor, storage, battery, sensors, and wireless interfaces so one compact device can manage apps and data on the move. Laptop is portable computer with an integrated display, keyboard, battery, and mobile-oriented internal components. It provides a full personal computing platform that can run unplugged and travel easily. Mechanically, it combines processor, memory, storage, battery, display, and wireless hardware in one chassis so the system operates as one endpoint. Docking Station is accessory that expands a portable computer with extra ports, displays, power, or peripherals through one connection. It lets a mobile system switch quickly between portable and desk-based work setups. Mechanically, it carries power, video, USB, and network traffic through one dock link so attached peripherals reconnect as a group. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
