## 2. Mini lecture
This is the review day for Wireless, SOHO design, cloud, and network security basics. The goal is to connect the week's main ideas into one working model rather than introduce a new topic. The core terms are SSID, Wi-Fi Standard, WPA3, Access Point, SOHO Router, and Firewall.

Across the week, the key mechanisms were: SSID: advertises the network name in beacon frames so clients can discover and join the correct wireless LAN Wi-Fi Standard: defines channel use, modulation, frame handling, and negotiated features so compatible devices can associate and exchange traffic WPA3: uses modern authentication and key establishment to protect wireless sessions against interception and offline guessing Access Point: advertises a network name, authenticates clients, and forwards their traffic between radio channels and the wired switch path The value of the review is in seeing how these pieces connect across setup, operation, failure, and troubleshooting.
