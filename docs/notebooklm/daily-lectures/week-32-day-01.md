The main terms in scope are Spaced Repetition and Retrieval Practice. The focus is on what each one does, how it operates, and how it affects the rest of the system.

Spaced Repetition is review technique that revisits material at increasing intervals within structured review and exam preparation. It schedules reviews at expanding intervals based on difficulty and prior recall performance. Mechanically, it schedules reviews at increasing intervals, bringing material back near the point it would otherwise be forgotten. Retrieval Practice is actively recalling information without notes to strengthen memory within structured review and exam preparation. It forces recall from memory without prompts, strengthening retention and identifying weak recall paths. Mechanically, it forces recall from memory first, then checks the answer so weak traces are strengthened through correction. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
