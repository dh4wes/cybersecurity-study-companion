## 2. Mini lecture
The week focus is Google Course 5: Assets, threats, and vulnerabilities (part 1), and today's objective is kick off the week's topic and build the right vocabulary. The main terms in scope are Asset Inventory and Data Classification. The focus is on what each one does, how it operates, and how it affects the rest of the system.

Asset Inventory is structured list of hardware, software, and data assets in scope. It maps systems, software, owners, and data to known records so investigations and control scope stay accurate. Mechanically, it collects host attributes, ownership, software, and data scope into records that other controls use for targeting and scoping. Data Classification is labeling data by sensitivity and handling requirements within web, data, and cloud services. It assigns sensitivity labels that determine storage, access, transmission, retention, and disposal controls. Mechanically, it assigns labels by sensitivity and handling need so storage, sharing, and retention rules can be enforced. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
