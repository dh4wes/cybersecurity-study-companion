# LAN and WAN in context

## 1. Today’s concept in plain language

## 2. Mini lecture
The week focus is Networking foundations: models, media, devices, and topologies, and today's objective is kick off the week's topic and build the right vocabulary. The main terms in scope are LAN and WAN. The focus is on what each one does, how it operates, and how it affects the rest of the system.

LAN is local area network that connects devices across a limited physical space such as a room, office, or campus. It provides high-speed local communication among nearby systems and shared resources. Mechanically, it uses local switching and addressing so devices exchange frames directly across a contained network segment. WAN is wide area network that connects distant sites across provider infrastructure or large geographic areas. It carries traffic between remote networks that cannot be joined by one local switching domain. Mechanically, it uses routed links across carrier or internet paths so separate local networks can exchange traffic over distance. Together, these terms describe where work happens, how state changes, and what the visible outcome looks like when the system is operating normally.
