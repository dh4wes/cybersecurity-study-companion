## 4. Key distinctions and common confusions
- Capability Gap vs Maturity Model — Capability Gap: Difference between current skills/process maturity and target state. Maturity Model: Framework used to assess current capability level and improvement targets.
- Purpose vs mechanism — be able to separate what Capability Gap is for from how it actually performs that job.
- Recognition vs diagnosis — knowing the term name is not enough; know what evidence would make you pick it in a scenario.
