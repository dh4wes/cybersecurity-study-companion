Modem means device that connects a local network to an ISP service by converting signals between provider media and Ethernet. Its practical job is that it provides the handoff between the provider network technology and the customer router or gateway. Mechanically, it receives the provider signal format, translates it into local data frames, and passes traffic between the ISP link and home equipment. Bluetooth means short-range wireless standard used to connect nearby devices such as headsets, keyboards, and phones. Its practical job is that it provides low-power local connectivity for peripherals and quick device-to-device data exchange. Mechanically, it pairs nearby radios, negotiates capabilities, and then exchanges data over a short-range wireless link. IP Address means logical network address that identifies a source or destination host for routed traffic. Its practical job is that it lets packets be sent toward the correct network and endpoint instead of staying on one local segment. Mechanically, it combines network and host portions so devices can tell whether traffic stays local or must be forwarded by a router. ## 4. Key distinctions and common confusions
- Modem vs Router — Modem: Device that connects a local network to an ISP service by converting signals between provider media and Ethernet. Router: Layer 3 device that forwards packets between networks.
- Bluetooth vs Wi-Fi Standard — Bluetooth: Short-range wireless standard used to connect nearby devices such as headsets, keyboards, and phones. Wi-Fi Standard: Wireless LAN specification that defines how clients and access points communicate over radio.
- IP Address vs MAC Address — IP Address: Logical network address that identifies a source or destination host for routed traffic. MAC Address: Hardware identifier used by network interfaces on local segments.
- Recognition vs diagnosis — knowing the term name is not enough; know what evidence would make you pick it in a scenario.
