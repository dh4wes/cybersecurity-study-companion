## 2. Mini lecture
The week focus is Wireless, SOHO design, cloud, and network security basics, and today's objective is kick off the week's topic and build the right vocabulary. The main terms in scope are SSID, Wi-Fi Standard, and WPA3. The focus is on what each one does, how it operates, and how it affects the rest of the system.

SSID is wireless network name advertised by an access point. It advertises or identifies a wireless network name that clients use during association. Mechanically, it advertises the network name in beacon frames so clients can discover and join the correct wireless LAN. Wi-Fi Standard is wireless LAN specification that defines how clients and access points communicate over radio. It sets compatibility, throughput, channel behavior, and security capabilities for wireless networking. Mechanically, it defines channel use, modulation, frame handling, and negotiated features so compatible devices can associate and exchange traffic. WPA3 is modern Wi-Fi security standard with stronger protections than earlier protocols. It protects wireless access using stronger authentication and encryption mechanisms than prior Wi-Fi standards. Mechanically, it uses modern authentication and key establishment to protect wireless sessions against interception and offline guessing.
