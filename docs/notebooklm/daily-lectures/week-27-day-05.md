Lateral Movement is post-compromise activity where an attacker moves from one system to other reachable systems. It expands attacker access and increases the damage that one compromised account or host can cause. Mechanically, it uses stolen access, trust paths, or remote tools to pivot from the initial foothold into additional internal systems.
